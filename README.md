# Sales Business Intelligence & SQL Analytics

A complete end-to-end Business Intelligence project that analyzes a Global Sales dataset using **MySQL**, **Microsoft Excel**, and **Power BI** to uncover actionable business insights. This project demonstrates data cleaning, SQL-based analysis, dashboard development, and business reporting for data-driven decision making.

---

##  Project Overview

This project focuses on analyzing over **51,000+ sales records** to identify key business trends, customer behavior, product performance, regional sales, profitability, and operational efficiency.

The project follows a complete analytics workflow:

- Data Cleaning & Preparation
- SQL Query Development
- Business Analysis
- Interactive Power BI Dashboard
- Executive Business Insights Report

---

##  Objectives

- Analyze overall sales performance
- Identify the most profitable products
- Discover high-value customers
- Compare regional sales performance
- Evaluate category profitability
- Analyze discount strategies
- Generate business recommendations using data
